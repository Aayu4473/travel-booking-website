# 🌍 ExploreGlobe – Travel Booking Website

## Overview

ExploreGlobe is a simple travel booking website created using HTML, CSS, and JavaScript. The website allows users to explore travel destinations and book trips through an interactive booking form.

The main goal of this project was to design a modern travel landing page and implement basic booking functionality using JavaScript. The interface focuses on a clean layout, smooth navigation, and an attractive travel-themed design.

---

## Features

* Modern travel landing page
* Navigation menu with multiple sections
* Destination search interface
* Trip booking page
* Price calculation based on number of travelers
* Simple booking workflow leading to payment
* Responsive and visually appealing UI

---

## Pages Included

**Home Page**

* Hero section with travel background
* Search form for destinations
* Navigation links (Destinations, Deals, About, Contact)

**Booking Page**

* Destination selection
* Travel date input
* Number of travelers
* Price calculation button
* Proceed to payment option

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

These technologies were used to build the complete frontend interface and implement the booking interaction logic.
